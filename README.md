# wifibruteforcer

A simple flutter application to bruteforce wifi networks

# Installation in macOS:
```
git clone https://github.com/krishpranav/wifibruteforcer.git
cd wifibruteforcer
flutter config --enable-macos-desktop
flutter channel dev
flutter upgrade
flutter pub get
flutter run -d macos
```
